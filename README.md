Thinkadx-template
===============
## 技术栈(前端)

1. vue(vue-cli3创建)

2. UI框架 iview


## 使用git安装

~~~
1、git clone https://gitee.com/first_pig/thinkadx-template.git

安装所有依赖
2、npm install

3、修改配置信息copy一份.example.env并改名为.env,按照注释填写参数即可

启动项目
4.npm run serve OR vue ui

5、登录 账号:admin 密码:admin
~~~
