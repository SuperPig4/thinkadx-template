Thinkadx-template
===============
## 技术栈(前端)

> vue3 全家桶

> iview


## 使用git安装

~~~
1、git clone https://gitee.com/first_pig/thinkadx-template.git

安装所有依赖
2、npm install

3、修改配置信息copy一份.example.env并改名为.env,按照注释填写参数即可

启动vue3的ui界面
4、vue ui

5、点击任务->serve->启动->启动APP

6、默认账号密码 admin admin
~~~
