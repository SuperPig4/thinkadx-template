Thinkadx-template
===============
## 技术栈(前端)

> vue3 全家桶

> iview


## 使用git安装

~~~
1、git clone https://gitee.com/first_pig/thinkadx-template.git

安装所有依赖
2、npm install

3、打开 src/plugins/Cm.js 将 Config.apiDomain 的接口域名换成你搭建的thinkadx服务器端地址

启动vue3的ui界面
4、vue ui

5、点击任务->serve->启动->启动APP

6、默认账号密码 admin admin
~~~
